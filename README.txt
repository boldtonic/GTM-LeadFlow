GTM LEAD FINDER - Setup Instructions
=====================================

REQUIREMENTS
------------
- Python 3.x (check: python3 --version)


FIRST TIME SETUP (one time only)
--------------------------------
Open Terminal and run:

    pip3 install flask flask-cors requests pyyaml


HOW TO RUN
----------
1. Open Terminal
2. Navigate to this folder:

       cd /path/to/GTM-LEAD-FINDER

3. Start the server:

       python3 app.py

4. Open your browser to:

       http://localhost:5001


TO STOP
-------
Press Ctrl+C in the Terminal


API KEYS
--------
Already configured in app.py. To change them, edit lines 26-27.
